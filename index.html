Tentang

Halaman ini adalah template — edit konten, ganti link, dan duplikat bagian download-card untuk lebih banyak file. Untuk file besar (>100MB) gunakan Google Drive / Mega dan gunakan tombol mirror untuk arahkan ke sana.

Cara tambahkan file baru

  1. Salin seluruh <section id="download-card"></section> ke bawah dan ubah teks / link.
  2. Atau buat file baru: downloads/nama-file.html dan salin konten card ke file itu.
  3. Pastikan link mirror diarahkan ke penyimpanan yang kamu pilih.
